# Language Agnostic Chatbot Backend

## Overview
This project is a backend system for a language agnostic chatbot that allows users to interact in multiple languages. The system detects the language of user input, translates it to a common processing language, generates a response, and translates it back to the user's language.

## Features
- Multi-language support
- Automatic language detection
- Language translation
- Secure authentication using JWT
- PostgreSQL database integration

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- JWT Authentication
- REST APIs

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the server:

python main.py
